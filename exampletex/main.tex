\subsection{Założenia aplikacji}

Aplikacja ma za zadanie nauczyć użytkownika słówek z danego języka obcego, który jest wybrany przez użytkownika. Program od strony serwerowej został napisany w języku C\# z użyciem frameworka ASP.NET MVC z wykorzystaniem bazy danych Microsoft SQL. Od strony klienta użyty został HTML wraz z arkuszami styli stworzonymi w oparciu o style SASS. Uruchamiany kod Javascript był skompilowanym kodem S %ASDASDASD
